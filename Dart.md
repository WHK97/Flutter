# Dart
Dart를 시작하게 되면 메인함수를 가장 먼저 보게 된다
```
void main(){}
```
메인함수에서는 내가 쓴 코드가 호출이 된다. 만약 메인함수명을 다르게 바꾼다면 에러가 발생을 한다.
```
void Hello(){} // 에러: No main method found 
```

Dart에서 함수, 변수등 끝에는 ;을 꼭 붙여 줘야 한다. 일부러 ;안붙이는 경우가 있기 떄문에 ;를 필수로 붙여줘야한다.
```
void main(){
    print("Hello Wold!"); // O
    // print("Hello Wold!") // X

}
```
